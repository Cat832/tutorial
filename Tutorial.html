Variabelen

Variabelen defineer je zo:

var Je_variabele = 4;

De waarde is nu 4. In variabelen mogen geen spaties komen

Variabelen kun je ook zo defineren; maar dan is de waarde 'undefined'

var Je_variabele;

Je hebt 3 soorten waardes:

var Je_variabele;
Je_variabele = true
Je_variabele = false
Je_variabele = "Dit is een string"
Je_variabele = 5

Een boolean is 'true' of 'false'. (waar of niet waar) Een string zit altijd tussen "" of ''

Een integer is een getal. Je kan een variabele ook laten verschijnen.

var variabele = 1;
alert(variabele);

Arrays

Een array is een lijst; je kan er meerdere waardes in stoppen.

Een array is eigenlijk een soort variabele.

Zo defineer je een array:

var MijnArray = [1, 2, 3];

Zo verwijder je items en voeg je items toe:

var MijnArray = [1, 2, 3];
MijnArray.pop(); //[1, 2]
MijnArray.push(3); //[1, 2, 3]
MijnArray.shift(); //[2, 3]
MijnArray.unshift(1); //[1, 2, 3]
alert(MijnArray);

If & Else

Om te detecteren of iets waar is, gebruik je if.

if () {}

In de () typ je in wat je moet detecteren. In de {} als de () waar is, doe dit.

Stel: Je hebt een variabele, en je wilt weten of dat 5 is, zoja: alert "Yes!" Doe dat zo:

var MijnVar = 5;
if (MijnVar == 5){
alert("Yes!")}

Zo detecteer je andere dingen:

&& en

Goed, nu je de if kent, laten we naar de else gaan.

Stel: je wilt weten of 1 1 is. Zoja, alert 'Wiskunde is makkelijk!' Anders alert 'Geen wiskunde'

else{alert('Geen wiskunde')} //alerts 'Wiskunde is makkelijk!'

if (1 != 1){alert('Wiskunde is makkelijk!')}
else{alert('Geen wiskunde')}

Expressies

Expressies zijn stukjes code die waar (true) of onwaar (false) zijn.

Een expressie is bijvoorbeeld 1 + 1 == 2. Dit is true.

Zo gebruik je expressies:

if (1 + 1 == 2){alert('Basis wiskunde')}

Maar niet alleen ... + ... == ... is een expressie, ook dit zijn expressies:

Delen: gebruik /

alert(10 / 2) //5

Vermenigvuldigen: gebruik *

alert(20 * 1) //20

Resten: gebruik %

alert(21 % 2) //1

Aftrekken: gebruik -

alert(21 - 2) //19

Machtsverheffen: gebruik **

alert(3 ** 2) //9

Loops

Loops zijn herhalingen. Je hebt de for en de while loop

For

Een for loop schrijf je zo: for(doe je bij begin; als dit zo is, eindig de loop; bij het eind doe dit.)

Voor de duidelijkheid:

var i;
for(i = 0; i <= 3; i++){alert(i)}

TIP: i++ betekent eigenlijk i = i + 1

Dit zal alerten:

0
1
2
3

While

De while loop is geactiveert als het gene wat tussen de () waar (true) is

Voor de duidelijkheid:

We maken de for loop na.

var i = 1;
while(i <= 3){alert(i)
i++}

Dit is eigenlijk gewoon:

var i;
for(i = 0; i <= 3; i++){alert(i)}

Want hier alert het ook gewoon:

0
1
2
3

Functies

Bij alles, heeft een functie ook () en {}

Dit is een voorbeeld van een functie:

function alerts(message){alert(message)}

Als je nu ergens in de code alerts('HI!') heb je gewoon alert('HI!')
